Project Description

CyberSecurityChatBotGUI2 is a chatbot developed in C# using Windows Forms. The chatbot helps educate users about cybersecurity topics such as phishing, password safety, scams, suspicious links, and online safety.

 Software Required

* Visual Studio 2022
* .NET Framework
* Windows Operating System

 How to Run the Project

1. Download or clone the repository.
2. Open the CyberSecurityChatBotGUI2.sln file in Visual Studio.
3. Build the solution.
4. Run the project by clicking Start.
5. Interact with the chatbot through the GUI.

Features Implemented

* Windows Forms GUI
* Cybersecurity awareness chatbot
* Keyword recognition
* Memory feature (stores user's name)
* Sentiment detection
* Random responses
* Event-driven programming
